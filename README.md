A little example about ng2 with gulp task. For now just works in browser with
es6 compatibility.

Usage
====

Install npm packages
```shell
npm install
```

For copy assets
```shell
gulp copy
```

For compile typescript
```shell
gulp typescript
```

For run a static server
```shell
gulp server
```

For deploy 
```shell
gulp
```
